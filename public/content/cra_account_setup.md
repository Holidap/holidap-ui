# 1. Canada Revenue Agency (CRA) 계정이 필요한 이유

캐나다 국세청(CRA) 계정(**My Account**)은 개인 세무 정보를 온라인으로 안전하게 관리할 수 있는 서비스입니다. CRA 계정을 개설하면 다음과 같은 이점이 있습니다:

- 세금 환급 진행 상황 실시간 확인
- 과거 세금 신고 내역 열람
- GST/HST 크레딧, 각종 정부 혜택 신청 및 현황 확인
- RRSP(연금저축), TFSA(비과세계좌) 한도 조회
- CRA에서 발송하는 서류를 온라인으로 바로 수령 가능

# 2. CRA 웹사이트 접속

- 캐나다 국세청 공식 웹사이트([CRA My Account](https://www.canada.ca/en/revenue-agency/services/e-services/cra-login-services.html))에 접속합니다.
- 화면에서 **"Register for CRA My Account"** 버튼을 클릭해 가입을 시작합니다.

# 3. 로그인 방법 선택

CRA 계정은 두 가지 방식 중 하나로 개설할 수 있습니다:

- **Sign-In Partner**: 캐나다의 은행 계좌(예: RBC, TD, Scotiabank 등) 온라인 뱅킹 정보를 이용하여 로그인
- **CRA 사용자 ID 생성**: CRA 전용 ID와 비밀번호를 직접 만들어 로그인

# 4. 본인 인증 절차 진행

- SIN(사회보장번호), 우편번호, 생년월일 등의 개인 정보를 입력합니다.
- 과거 세금 신고 내역의 일부 정보를 입력하라는 화면이 나올 수 있습니다.
- 캐나다 세금 신고가 처음이라면, 이전 세금 신고 내역 입력 단계에서 "해당 없음(N/A)"을 선택하거나 "0"을 입력할 수 있지만, 이 경우 CRA가 추가 신원확인 절차를 진행할 수 있습니다.

# 5. 보안 코드 수령

CRA는 계정 개설 후 보안 강화를 위해 보안 코드를 받아야 합니다:

- **우편 발송**: 보통 5~10 영업일 정도 소요
- **온라인 인증**: 정부 발급 신분증(ID)을 사용해 온라인 인증 서비스를 연결하거나, 일부 주정부 앱(예: BC Services Card app)을 이용하면 보안 코드를 모바일로 즉시 수령

# 6. 보안 코드 입력 후 계정 활성화

받은 보안 코드를 CRA 사이트에 입력하면 계정이 최종 활성화되며, 이후 CRA의 모든 온라인 서비스를 이용할 수 있습니다.

# 7. 정리

CRA 계정을 개설하면 환급 진행 상황, 과거 신고 내역, RRSP 한도, GST/HST 크레딧 등의 정보를 온라인으로 편리하게 확인할 수 있어 매우 유용합니다. 캐나다에서 세무 관련 업무를 처리할 때 필수적인 도구이므로, 가능한 한 빨리 개설하는 것을 권장합니다.
